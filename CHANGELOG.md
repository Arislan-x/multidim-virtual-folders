# Changelog

## 0.1.1

- Add official Obsidian lint checks for release preparation.
- Raise the minimum Obsidian version to 1.7.2 because the plugin uses `revealLeaf`.
- Clean up asynchronous event handling for sidebar activation, file opening, drag-and-drop, and confirmation dialogs.
- Improve settings page compatibility with Obsidian's recommended setting heading pattern.
- Localize write mode labels in the settings page.

## 0.1.0

- Initial public release.
- Create unlimited virtual folder dimensions from note frontmatter.
- Show dimensions in the left sidebar, right sidebar, or both.
- Support date dimensions with year/month/day root modes.
- Support path dimensions with temporary virtual paths.
- Drag Markdown notes into virtual folders to update frontmatter.
- Rename, delete, create subcategories, and change icons from the context menu.
- Per-dimension icons, colors, sorting, display side, and write mode.
- Emoji and Lucide icon picker.
- Chinese and English interface.
